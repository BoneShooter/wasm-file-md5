## wasm calc md5 demo

### test wasm:

1. npm i

2. npm run server

3. then open url : http://127.0.0.1:8080/main.html

4. click "calc" button, see the result on page

### build wasm:

1. install emscripten first:

    emscripten document: https://emscripten.org/docs/getting_started/downloads.html

2. sh build.sh
